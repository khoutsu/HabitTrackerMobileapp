# Feature Wishlist & Improvements

จากการตรวจสอบแอปพลิเคชัน Loop Habit Tracker ปัจจุบัน พบว่าฟีเจอร์พื้นฐานครบถ้วน แต่สามารถเพิ่มฟีเจอร์ดังต่อไปนี้เพื่อยกระดับประสบการณ์ผู้ใช้ได้:

## 1. ระบบแจ้งเตือน (Reminders & Notifications)

Priority: High

- สถานะปัจจุบัน: ยังไม่มีระบบแจ้งเตือนในหน้าสร้างกิจวัตร
- รายละเอียด: เพิ่มตัวเลือกให้ผู้ใช้ตั้งเวลาแจ้งเตือน (เช่น แจ้งเตือนทุกวันตอน 8:00 น.) เพื่อกระตุ้นให้ทำกิจวัตร
- สิ่งที่ต้องทำ:
  - เพิ่ม Local Notifications
  - เพิ่ม UI เลือกเวลาใน `HabitFormScreen`

## 2. ค้นหาและเรียงลำดับ (Search & Sort)

Priority: Medium

- สถานะ: **เสร็จสิ้น (Done)**
- รายละเอียด: เพิ่มระบบค้นหา (Search Bar) และระบบเรียงลำดับ (Sort) ตามชื่อ, สี, วันที่สร้าง (แบบ Manual เป็นค่าเริ่มต้น)

## 3. บันทึกโน้ตรายวัน (Daily Notes / Reflection)

Priority: Medium

- สถานะ: **เสร็จสิ้น (Done)**
- รายละเอียด: เพิ่มฟิลด์ `note` ในระบบ และ UI สำหรับเพิ่ม/แก้ไขโน้ต (กดค้างที่การ์ดกิจวัตร)

## 4. จัดเก็บกิจวัตร (Archive Habit)

Priority: Low

- สถานะ: **เสร็จสิ้น (Done)**
- รายละเอียด: มีฟังก์ชัน Archive/Unarchive และหน้ารายการที่ถูกจัดเก็บเรียบร้อยแล้ว

## 5. ปรับปรุงหน้าสถิติ (Enhanced Statistics)

Priority: Low

- สถานะปัจจุบัน: มีกราฟวงกลมและแท่งพื้นฐาน
- รายละเอียด: เพิ่ม GitHub-style Heatmap (ตารางสีเข้ม-อ่อนตามความถี่) เพื่อให้เห็นภาพรวมทั้งปีได้สวยงามและชัดเจนขึ้น
- สิ่งที่ต้องทำ:
  - นำ `CalendarHeatmap` มาใช้ในหน้า Detail (ถ้ายังไม่มี หรือปรับปรุงให้ชัดเจนขึ้น)
