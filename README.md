# Loop Habit Tracker - Flutter App

A simple, beautiful, and open-source habit tracking application built with Flutter. This project aims to replicate the core functionality and design principles of the original "Loop Habit Tracker" Android app.

## ✨ Features

*   **Habit Management:** Create, edit, and delete habits with flexible schedules.
*   **Progress Tracking:** Visualize your progress with detailed statistics, calendar heatmaps, and strength charts.
*   **Motivation:** Track streaks and see your habit strength evolve.
*   **Data Privacy:** All data is stored locally on your device. No ads, no tracking.
*   **Customization:** Theme switching (Light/Dark/System) and multi-language support (English, Thai).
*   **Backup & Restore:** Export and import your data (CSV, SQLite).
*   **Home Screen Widgets:** Keep track of your habits directly from your home screen (Android only for now).
*   **Onboarding:** A guided tour for new users.

## 🚀 Getting Started

### Prerequisites

*   [Flutter SDK](https://flutter.dev/docs/get-started/install) installed and configured.
*   An IDE (e.g., VS Code, Android Studio) with Flutter and Dart plugins.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your_username/loop_habit_tracker.git
    cd loop_habit_tracker
    ```
2.  **Get dependencies:**
    ```bash
    flutter pub get
    ```
3.  **Run the app:**
    ```bash
    flutter run
    ```

### Running Tests

*   **Unit Tests:**
    ```bash
    flutter test test/unit_test.dart
    ```
*   **Widget Tests:**
    ```bash
    flutter test test/widget_test.dart
    ```
*   **Integration Tests:**
    ```bash
    flutter test integration_test/app_test.dart
    ```
